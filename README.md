# dapp-meetup
 CSC2125 Project

[documentation](https://leonharde.github.io/documents/CSC2125_Report.pdf)

## Usage

* Start Ganache, MetaMask

* deploy contract

  ```
  truffle migrate --reset
  ```

* run app

  ```
  cd client && npm start
  ```
